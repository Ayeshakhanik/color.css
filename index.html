COLOR

Why do we use colors?

  • Make the website attractive.
  • Improve readability.
  • Highlight important content.
  • Organize information clearly.
  • Make the design more professional.

TYPES OF COLORS

  1. FOREGROUND COLOR
  2. BACKGROUND COLOR

DIFFERENT WAYS TO WRITE COLORS

  1. COLOR NAME
  2. RGB VALUES
  3. HEXADECIMAL COLOR CODE

COLOR NAME

Uses predefined color names like red, blue, green, and yellow.
Easy to read and write.
Best for simple web designs.

RGB (Red, Green, Blue)

Defines colors by combining Red, Green, and Blue values.
Each value ranges from 0 to 255.

Hexadecimal (Hex Code)

Represents colors using a 6-digit code starting with #.
Uses numbers (0–9) and letters (A–F).
